# 🛒 Sistema de Gestão de Loja

Este é um projeto desenvolvido em **Python** para a disciplina de Programação. O objetivo é fornecer uma ferramenta simples e eficiente para gerir o inventário de uma pequena loja.

---

## 📋 Planeamento (Fase 1)

* **Nome do Projeto:** Gestão de Loja TechPoint
* **Objetivo:** Controlar a entrada, saída e consulta de produtos em stock.
* **Dados Guardados:**
    * `Nome`: Identificação do produto.
    * `Preço`: Valor unitário em Euros (€).
    * `Quantidade`: Total de unidades em stock.

---

## 🛠️ Funcionalidades

O programa utiliza um menu interativo baseado num ciclo `while` com as seguintes opções:

1.  **Adicionar:** Regista novos produtos numa lista de dicionários.
2.  **Listar:** Exibe todos os produtos, preços e quantidades atuais.
3.  **Procurar:** Localiza um produto específico pelo nome.
4.  **Remover:** Elimina um produto do sistema.
5.  **Estatísticas (Extra):** Calcula o valor total do inventário e a quantidade total de itens.
0.  **Sair:** Encerra o programa com segurança.
